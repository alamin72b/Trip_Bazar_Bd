# Prechecks

Before merging into `develop`, confirm:

- the current branch is not `develop`
- the current branch is not `main`
- the current branch name starts with `feature/`, `fix/`, `docs/`, or `chore/`
- the working tree is clean before switching branches
- the user intends normal task-branch integration into `develop`

If any of these checks fail, stop and explain the reason clearly.
